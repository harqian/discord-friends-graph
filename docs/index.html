Discord Friends Graph

Map your Discord friends list as an interactive mutual-connections graph. Runs entirely in your browser. No backend, no accounts, no data uploaded.

Install for Chrome ↗ Source ↗

Make your own graph

  1. Install the extension.
  2. Open discord.com while logged in.
  3. Click the extension icon → Scan Friends.
  4. Hit View Graph.

A scan takes about 1 second per friend. Stop anytime.

Merge with someone else's

  1. Make your own graph first.
  2. Open the page that hosts theirs (or download its .html).
  3. Click the icon → Merge with shared page → Read active tab.
  4. Preview the union, optionally publish a combined share.

Works on embedded iframes too. You see how many of their friends are also yours.

See an example

A five-friend sample graph. Pan, zoom, click a node, press / to search.

What about privacy?
  • The extension makes no requests to any server we operate.
  • Your scan stays in chrome.storage.local on your machine.
  • When you publish a share, you choose what to hide: names, avatars, profile links, or specific friends.
  • Discord user IDs always remain readable in published shares — that's what makes merging possible. To exclude someone entirely, use the omit option.
  • The merge flow uses activeTab: the extension only reads the current tab when you click its icon.

Full policy: PRIVACY_POLICY.md.

How to share + embed

From the popup, hit Export Shareable Page. You get a self-contained .html file you can host anywhere. Embed with:

<iframe src="your-share.html" width="100%" height="600" style="border:0"></iframe>

Every embedded share has built-in Download, Copy embed, and (when in an iframe) Open in tab buttons. Visitors can save the file or grab the iframe snippet themselves.
